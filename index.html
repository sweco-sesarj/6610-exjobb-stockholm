    • Revit Manager Robot

      ​​​Problem:
      Idag finns det ingen smidig metod/program för att hantera flera olika Revitmodeller simultant. I ett stort projekt där man har flera modeller och som man projekterar samtidigt, så går det inte att ändra vyinställningarna, Legender, Filter, exporter osv i samtliga modeller samtidgt.
      Efterfrågan:
      Helst ett program (Robot, ett gränssnitt utanför Revit) som kan hjälpa BIM-ansvarige att hantera flera modeller samtidigt. Helst inget plugIn i revit, vad som föredras är ett program utaför revit med ett enkelt gränssnitt som hjälper en att styra flera revitmodeller samtidigt.
      Rekommenderad Metod:
      Ett samarbet mellan Systemutvecklarna och BIMIS. Personen får tekniskstöd av Systemutvecklarna och få råd, vägledning av BIMIS.
      Förväntad Resultat:
      Ett program som alla BIM-ansvarige i Sweco kan ta hjälp av för sin Revitmodellhantering i olika projekt. Detta minskar antalet fel pga den mänskliga-faktorn samt kvalitetsäkrar revitmodellerna för leverans till BH/RH. Detta leder framförallt till att man effektiviserar arbetet och lägger tiden på nyttigare arbetsuppgifter
